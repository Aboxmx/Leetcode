## 递归算法的设计

1. 明确函数的功能

   * 先不去思考里面代码怎么写，首先搞清楚这个函数干嘛用的，能完成什么功能

2. 明确原问题与子问题的关系

   * 寻找f(n)与f(n-1)的关系

3. 明确递归基（边界条件）

   * ~~~c++
     int sum(int n) { //1.函数功能：求n个数的和
         
         sum(n-1); //2.子问题
             
         return sum(n-1) + n; //3.子问题和原问题的关系
         
         if (n <= 1) {
             return n; //4.明确递归基
         }
     }
     ~~~
