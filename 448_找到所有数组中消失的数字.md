## 找到所有数组中消失的数字

1. 遍历数组，对出现的元素，在对应的下标-1的位置，将该元素置为负的，若已经为负，则跳过
2. 遍历数组，若该元素不为负，则将其下标添加到res中

~~~java
class Solution {
    public List<Integer> findDisappearedNumbers(int[] nums) {
        if (nums == null ) {
            return null;
        }
        List<Integer> res = new ArrayList<>();
        int len = nums.length;
        for (int i = 0; i < len; i++) {
            int newIndex = Math.abs(nums[i]) - 1;
            if (nums[newIndex] > 0) {
                 nums[newIndex] *= -1;
            }

        }
        for (int i = 0; i < len; i++) {
            if (nums[i] > 0) {
                res.add(i + 1);
            }
        }
        return res;
    }
}
~~~
