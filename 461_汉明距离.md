## 汉明距离

1. number & number-1的值即为new，该值的二进制相比于number，最右边的1变为0
2. 当xor = x ^ y的值不为零时，循环
3. 每次计数加一，并且更新xor = xor & xor-1

~~~java
class Solution {
    public int hammingDistance(int x, int y) {
        int xor = x ^ y;
        int count = 0;
        while (xor != 0) {
            count++;
            xor &= xor - 1;
        }
        return count;
    }
}
~~~
