## 最大子序和（模板）

### 贪心算法

贪心策略：当前和为最大和，若和为负数，则更新为0

1. 设置理论最大值时，用最小值
2. 遍历数组，枚举每一个元素
3. 动态的计算当前和
4. 判断，若当前和大于最大和，则更新
5. 判断，若当前和小于0，则重新计算当前和

~~~java
class Solution {
    public int maxSubArray(int[] nums) {
        int len = nums.length;
        int thisSum = 0, maxSum = Integer.MIN_VALUE;
        for (int i = 0; i < len; i++) {
            thisSum += nums[i];
            if (thisSum > maxSum) {
                maxSum = thisSum;
            }
            if (thisSum < 0) {
                thisSum = 0;
            }
        }
        return maxSum;
    }
}
~~~
