## 爬楼梯

动态规划且只需记录最新状态

1. 初始化开始状态1
2. 到达n的状态，需要做n-1步，所以循环次数为n-1
3. 循环n-1次，动态更新状态

~~~java
class Solution {
    public int climbStairs(int n) {   
        int first = 1;
        int second = 2;
        for(int i = 1; i < n; i++) {
            second = first + second;
            first = second - first;
        }
        return first;
    }
}
~~~
