## 移动零：双指针

1. 遍历数组
2. 判断当前元素，若不为0，则与指针j所指向的元素互换

~~~java
class Solution {
    public void moveZeroes(int[] nums) {
        if (nums == null || nums.length == 0) {
            return;
        }
        int len = nums.length;
        int j = 0; //指向下一个存放非零元素的位置
        for (int i = 0; i < len; i++) {
            if (nums[i] != 0) {
                int tmp = nums[i];
                nums[i] = nums[j];
                nums[j++] = tmp;
            }
        }
    }
}
~~~
