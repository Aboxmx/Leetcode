## 只出现一次的数字

### 解法：使用异或运算

1. a^0 = a
2. a^a = 0
3. a^b^a = b

* 遍历整个数组，初始值设为0
* 累计异或，最终结果即为出现一次的数字

~~~java
class Solution {
    public int singleNumber(int[] nums) {
        int single = 0;
        for (int num : nums) {
            single ^= num;
        }
        return single;
    }
}
~~~
